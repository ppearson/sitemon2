Sitemon

Sitemon is an integrated website monitoring and load-testing tool for websites.

It can run scheduled tests against a website, simulating an end-user using the website.

It can run performance tests against a website, simulating hundreds of users using a website at once.

It is cross-platform (Windows, OS X and Linux), and is written in C++.

See sitemon_manual.pdf in /documentation for details on how to use it.
